Limbajul HTML:

HTML(HyperText Markup Language) este un limbaj de marcare utilizat pentru a crea pagini web.

Un exemplu de cod html îl găsiți în imaginea următoare care se află la adresa http://upload.wikimedia.org/wikipedia/commons/e/e4/HTML_logo.png .

Imaginea are atributul height de valoare 147.



Tag-urile html cele mai utilizate în realizarea tabelelor sunt:

table definește un tabel
tr un rând într-un tabel
td o celulă într-un tabel


Observație: Pentru a pune un contur unui tabel se poate (dar nu se recomandă) să se pună atributul border="1".
La validare va da eroarea:The border attribute on the table element is obsolete.

Formularele se pot crea cu ajutorul tag-ului form care poate conține tag-urile:input, label, select, option, button.
Pentru mai multe detalii consultați cursul sau tutorialul HTML Forms and Inputs de la adresa http://www.w3schools.com/html/html_forms.asp

Sunt de acord cu condițiile

Pentru a crea liste se pot folosi tag-urile:

  1. ul - listă neordonata
  2. ol - listă ordonata
  3. li - item din listă
  4. dl - listă cu descrieri
  5. dt - termen dint-o listă cu descrieri
  6. dd - descrierea dintr-o listă cu descrieri


Pentru a insera un document HTML în altul se poate folosi tag-ul <iframe>.
Astfel, exemplul de mai jos insereaza conținutul de la adresa https://html.spec.whatwg.org/multipage/.
